# 📚 Student Attendance Management System

A simple and efficient **web-based Student Attendance Management System** built using **HTML, CSS, JavaScript (and React if applicable)**.  
This project helps manage student attendance digitally with an easy-to-use interface and keeps track of attendance history.

---

## 🚀 Features

- 👨‍🎓 Add and manage student records  
- 📝 Mark daily attendance (Present / Absent)  
- 📅 Attendance history for each student  
- 💾 Data stored using Local Storage (or database if used)  
- 📊 Simple and user-friendly interface  
- 🔍 View attendance records anytime  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  
- JavaScript (ES6)  
- React (if used)  
- Local Storage / JSON
